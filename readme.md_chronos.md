# Chronos — Sistema de Gestão de Ponto para Pesquisa Acadêmica

Chronos é uma solução de marcação de ponto com UX de nível enterprise, desenvolvida especificamente para laboratórios de pesquisa universitários. O sistema foi projetado para atender bolsistas de Iniciação Científica (IC) e Pós-Graduação (PG), priorizando rapidez, confiabilidade e uma interface sofisticada.

## 🚀 Visão Geral

O projeto foca em uma experiência de usuário extremamente limpa e intuitiva, inspirada em grandes referências de design moderno como **Stripe**, **Linear** e **Apple Human Interface**. A premissa central é reduzir a carga cognitiva, permitindo que o pesquisador registre sua jornada com apenas um toque.

## ✨ Funcionalidades Principais

- **Marcação de Ponto Inteligente:** Botão dinâmico que altera seu estado (Entrada, Almoço, Retorno, Saída) automaticamente conforme o fluxo do dia.
- **Relógio de Alta Precisão:** Visualização em tempo real da jornada atual.
- **Dashboard de Progresso:** Acompanhamento visual da meta diária de carga horária.
- **Histórico Detalhado:** Visualização mensal de registros com status de conclusão da carga horária.
- **Perfil Acadêmico:** Gestão de informações como orientador, laboratório e carga horária semanal contratada.

## 🎨 Design System

O projeto utiliza o sistema **Academic Precision**, focado em clareza e hierarquia visual:
- **Paleta:** Azul Profissional (#2563EB) como cor primária, complementada por tons de superfície claros (#F8FAFC).
- **Tipografia:** Inter (Sans-serif) para máxima legibilidade.
- **Formas:** Bordas arredondadas de 16px a 20px para uma aparência moderna e amigável.
- **Modo:** Otimizado para Light Mode (Dark Mode planejado).

## 🛠️ Tecnologias

- **Frontend:** HTML5, Tailwind CSS.
- **Ícones:** Material Symbols (Google).
- **UX/UI:** Mobile-first, totalmente responsivo para Android e iOS.

## 📂 Estrutura do Repositório

```text
├── index.html          # Tela de Login
├── cadastro.html       # Cadastro de novos pesquisadores
├── ponto.html          # Tela Principal de Marcação (Dashboard)
├── historico.html      # Relatórios e registros passados
└── perfil.html         # Configurações do usuário
```

## ⚙️ Como Executar

Como o projeto é desenvolvido com HTML puro e Tailwind CSS (via CDN), basta abrir qualquer um dos arquivos `.html` diretamente no seu navegador de preferência.
